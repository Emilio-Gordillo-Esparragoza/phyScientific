# Contributing to LectureLab

Thanks for your interest. This project is licensed under the **Apache License 2.0** — see [`LICENSE`](LICENSE).

## Ways to contribute

- Improve docs or pedagogical wording in the Streamlit labs
- Add tests, bug fixes, or clearer math write-ups under `docs/`
- Extend dataset extracts / catalogs for more The Well ensembles
- Report issues with steps to reproduce

## Pull requests

1. Fork [LectureLab](https://github.com/Emilio-Gordillo-Esparragoza/LectureLab) and create a branch from `main`
2. Keep changes focused; match existing code style
3. Open a PR describing **why** the change helps learners or maintainers
4. By submitting a contribution, you agree it is licensed under Apache 2.0

## Local setup

See the [README](README.md) for Python env, Streamlit, and data notes.

Educational reuse (teaching, workshops, course materials) is explicitly welcome under Apache 2.0 — keep the license and copyright notice.
